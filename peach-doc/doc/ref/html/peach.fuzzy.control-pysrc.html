 10  __doc__ = """ 
 11  This package implements fuzzy controllers, of fuzzy inference systems. 
 12   
 13  There are two types of controllers implemented in this package. The Mamdani 
 14  controller is the traditional approach, where input (or controlled) variables 
 15  are fuzzified, a set of decision rules determine the outcome in a fuzzified way, 
 16  and a defuzzification method is applied to obtain the numerical result. 
 17   
 18  The Sugeno controller operates in a similar way, but there is no defuzzification 
 19  step. Instead, the value of the output (or manipulated) variable is determined 
 20  by parametric models, and the final result is determined by a weighted average 
 21  based on the decision rules. This type of controller is also known as parametric 
 22  controller. 
 23  """
