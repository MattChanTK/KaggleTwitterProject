 11  Radial Basis Function Networks 
 12   
 13  This sub-package implements the basic behaviour of radial basis function 
 14  networks. This is a two-layer neural network that works as a universal function 
 15  approximator. The activation functions of the first layer are radial basis 
 16  functions (RBFs), that are symmetric around the origin, that is, the value of 
 17  this kind of function depends only on the distance of the evaluated point to the 
 18  origin. The second layer has only one neuron with linear activation, that is, it 
 19  only combines the inputs of the first layer. 
 20   
 21  The training of this kind of network, while it can be done using a traditional 
 22  optimization technique such as gradient descent, is usually made in two steps. 
 23  In the first step, the position of the centers and the width of the RBFs are 
 24  computed. In the second step, the weights of the second layer are adapted. In 
 25  this module, the RBFN architecture is implemented, allowing training of the 
 26  second layer. Centers must be supplied, but they can be easily found from the 
 27  training set using algorithms such as K-Means (the one traditionally used), 
 28  SOMs or Fuzzy C-Means. 
 29  """

243          Presents a training set to the network.

245          This method automatizes the training of the network. Given a training 
246          set, the examples are shown to the network (possibly in a randomized 
247          way). A maximum number of iterations or a maximum admitted error should 
248          be given as a stop condition. 
249   
250          :Parameters: 
251            train_set 
252              The training set is a list of examples. It can have any size and can 
253              contain repeated examples. In fact, the definition of the training 
254              set is open. Each element of the training set, however, should be a 
255              two-tuple ``(x, d)``, where ``x`` is the input vector, and ``d`` is 
256              the desired response of the network for this particular input. See 
257              the ``learn`` and ``feed`` for more information. 
258            imax 
259              The maximum number of iterations. Examples from the training set 
260              will be presented to the network while this limit is not reached. 
261              Defaults to 2000. 
262            emax 
263              The maximum admitted error. Examples from the training set will be 
264              presented to the network until the error obtained is lower than this 
265              limit. Defaults to 1e-5. 
266            randomize 
267              If this is ``True``, then the examples are shown in a randomized 
268              order. If ``False``, then the examples are shown in the same order 
269              that they appear in the ``train_set`` list. Defaults to ``False``. 
270          ''' 
271          i = 0 
272          error = 1 
273          s = len(train_set) 
274          while i<imax and error>emax: 
275              if randomize: 
276                  x, d = random.choice(train_set) 
277              else: 
278                  x, d = train_set[i%s] 
279              error = self.feed(x, d) 
280              i = i+1 
281          return error
