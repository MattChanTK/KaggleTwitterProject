Class RaisedCosine

source code

Raised Cosine function.

Given the center and the frequency, creates a function that is a period of a raised cosine, that is:

0, if x <= xm - pi/w or x > xm + pi/w;

0.5 + 0.5 * cos(w*(x - xm)), if xm - pi/w <= x < xm + pi/w;
