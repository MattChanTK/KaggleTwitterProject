 10  __doc__ = """ 
 11  Base activation functions and base class 
 12   
 13  Activation functions define if a neuron is activated or not. There are a lot of 
 14  different definitions for activation functions in the literature, and this 
 15  sub-package implements some of them. An activation function is defined by its 
 16  response and its derivative. Being conveniently defined as classes, it is 
 17  possible to define a custom derivative method. 
 18   
 19  In this package, also, there is a base class that should be subclassed if you 
 20  want to define your own activation function. This class, however, can be 
 21  instantiated with a standard Python function as an initialization parameter, and 
 22  it is adjusted to work with the internals of the package. 
 23   
 24  If the base class is instantiated, then the function should take a real number 
 25  as input, and return a real number. The response of the function determines if 
 26  the neuron is activated or not. 
 27  """
