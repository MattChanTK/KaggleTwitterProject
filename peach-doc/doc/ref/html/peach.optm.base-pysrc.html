 32      Creates a function that computes the gradient vector of a scalar field. 
 33   
 34      This function takes as a parameter a scalar function and creates a new 
 35      function that is able to compute the derivative (in case of single variable 
 36      functions) or the gradient vector (in case of multivariable functions. 
 37      Please, note that this function takes as a parameter a *function*, and 
 38      returns as a result *another function*. Calling the returned function on a 
 39      point will give the gradient vector of the original function at that point:: 
 40   
 41        >>> def f(x): 
 42                return x^2 
 43   
 44        >>> df = gradient(f) 
 45        >>> df(1) 
 46        2 
 47   
 48      In the above example, ``df`` is a generated function which will return the 
 49      result of the expression ``2*x``, the derivative of the original function. 
 50      In the case ``f`` is a multivariable function, it is assumed that its 
 51      argument is a line vector. 
 52   
 53      :Parameters: 
 54        f 
 55          Any function, one- or multivariable. The function must be an scalar 
 56          function, though there is no checking at the moment the function is 
 57          created. If ``f`` is not an scalar function, an exception will be 
 58          raised at the moment the returned function is used. 
 59        dx 
 60          Optional argument that gives the precision of the calculation. It is 
 61          recommended that ``dx = sqrt(D)``, where ``D`` is the machine precision. 
 62          It defaults to ``1e-5``, which usually gives a good estimate. 
 63   
 64      :Returns: 
 65        A new function which, upon calling, gives the derivative or gradient 
 66        vector of the original function on the analised point. The parameter of 
 67        the returned function is a real number or a line vector where the gradient 
 68        should be calculated. 
 69      ''' 
 70      def _df(x, dx=dx): 
 71          if isscalar(x): 
 72              x = float(x) 
 73              return (f(x+dx) - f(x-dx)) / (2.*dx) 
 74          else: 
 75              n = x.size 
 76              df = zeros((n, )) 
 77              for i in xrange(n): 
 78                  xl = array(x) 
 79                  xl[i] = xl[i] - dx 
 80                  xr = array(x) 
 81                  xr[i] = xr[i] + dx 
 82                  df[i] = (f(xr) - f(xl)) / (2.*dx) 
 83              return df

178              is dependent of the technique implemented; 
179            - ``emax`` is the maximum allowed error. The default value above is

189          method signature is:: 
190   
191            step(self) 
192

195          solution and the estimate of the error. 
196

198          Implement this method to restart the optimizer. An optimizer might be 
199          restarted for a number of reasons: to escape a local minimum, to try
